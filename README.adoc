= JAX-RS MessageBodyReader/Writer for YAML

This project provides a JAX-RS `MessageBodyReader` and `MessageBodyWriter` for `application/yaml`.

Please note that, as with all deserialization frameworks, it may be unsafe to deserialize YAML from untrusted sources.
So you'll have to enable the `MessageBodyReader` by https://microprofile.io/specifications/config/[configuring] `yaml.unsafe-read.enabled` to `true`.

Under the hood, it uses the venerable https://bitbucket.org/snakeyaml/snakeyaml[SnakeYAML] (for now).
